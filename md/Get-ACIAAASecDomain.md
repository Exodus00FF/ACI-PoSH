# Get-ACIAAASecDomain

## SYNOPSIS
Get all AAA Security Domains for the Fabric

## SYNTAX

```
Get-ACIAAASecDomain [<CommonParameters>]
```

## DESCRIPTION
Get all Fabric AAA Security Domains defined within the fabric.
