# Get-ACIAAALocalUsers

## SYNOPSIS
Get all AAA Local Users for the Fabric

## SYNTAX

```
Get-ACIAAALocalUsers [<CommonParameters>]
```

## DESCRIPTION
Get all Fabric AAA Local Users defined within the fabric.

## EXAMPLES

### EXAMPLE 1
```
Get-ACIAAALocalUsers
```
